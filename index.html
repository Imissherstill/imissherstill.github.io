Daily 9:30pm reminder

iOS won’t let a web app fire a scheduled notification on its own. The reliable way is a built-in Shortcuts automation — about 30 seconds, works forever:
  1. Open the Shortcuts app → Automation tab
  2. Tap + → Time of Day → set 9:30 PM, Daily
  3. Choose Run Immediately
  4. Add action Open App → pick Momentum
Every night at 9:30 it opens the app so you log screen time and lock in the day.
